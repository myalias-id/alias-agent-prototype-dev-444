🤖 Embedded Chatbot Widget

This page demonstrates the modern chatbot widget with floating button and popup interface.

How it works: A floating logo button appears at the bottom-right corner. Click it to open the chat popup, and click again (or use the close button) to close it.

Features

  • Floating logo button with smooth pulse animation
  • Click to toggle chat popup with smooth animations
  • Close button inside chat header
  • Fully responsive design for mobile devices
  • Modern UI with shadows and transitions

Usage


<!-- Load the embed script -->
<script src="https://your-agent-link.com/embed.js"></script>

<!-- Initialize -->
<script>
  EmbeddedChat.init({
    width: '380px',    // Optional: popup width (default: 380px)
    height: '600px',   // Optional: popup height (default: 600px)
    borderRadius: '16px', // Optional: border radius (default: 16px)
    logoUrl: '/embed-logo.png', // Optional: logo path (default: /embed-logo.png)
    buttonSize: '60px'   // Optional: button size (default: 60px)
  });   
</script>
